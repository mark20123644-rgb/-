#  Telegram Bot для заказа ботов и рефералов

Этот бот помогает пользователям заказывать разработку ботов, оставлять отзывы и участвовать в реферальной программе. Администраторы могут делать рассылки и отвечать пользователям.

##  Возможности

- Реферальная система с начислением бонусов
- Приём заказов на разработку ботов
- Сбор отзывов (отправка админам и в Telegram-канал)
- Автоматический ежедневный прогрев
- Админ-команды: `/broadcast` (рассылка), `/reply` (ответ пользователю)

##  Системные требования

- **Linux Mint** (или любой дистрибутив Linux)
- Python 3.10 или выше
- Терминал

##  Быстрый старт
- В **Linux Mint** надо только создать .env открой терминал, перейди в папку, куда ты скачал или склонировал бота (например, cd my_bot), а затем выполни chmod +x run.sh. запусти run.sh
